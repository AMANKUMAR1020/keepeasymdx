# Stack

> A **stack** is a linear data structure that follows the **Last In, First Out (LIFO)** principle. The last element added is the first to be removed.

---

## When to use it

- **Undo/redo functionality** (editors, IDEs)
- **Function call management** (call stack)
- **Balanced parentheses**, **expression evaluation**
- **Backtracking algorithms** (DFS, maze solving)

---

## Visual: Stack behavior

```plaintext
Stack (Top at the right):

Initial:     []
Push 10:     [10]
Push 20:     [10, 20]
Push 30:     [10, 20, 30]
Pop():       [10, 20] → returns 30
Top():       20 (without removing)
```

<codeblock>
  #include <iostream>
#include <vector>

class Stack {
private:
    std::vector<int> data;

public:
    // Push element onto the stack
    void push(int value) {
        data.push_back(value);
    }

    // Remove and return the top element
    int pop() {
        if (data.empty()) {
            std::cerr << "Stack Underflow\n";
            return -1;
        }
        int top = data.back();
        data.pop_back();
        return top;
    }

    // Return the top element without removing it
    int top() const {
        if (data.empty()) {
            std::cerr << "Stack is empty\n";
            return -1;
        }
        return data.back();
    }

    // Check if the stack is empty
    bool empty() const {
        return data.empty();
    }

    // Return the size of the stack
    int size() const {
        return data.size();
    }
};

int main() {
    Stack s;
    s.push(10);
    s.push(20);
    s.push(30);

    std::cout << "Top: " << s.top() << "\n";    // 30
    std::cout << "Pop: " << s.pop() << "\n";    // 30
    std::cout << "Top after pop: " << s.top() << "\n"; // 20
    return 0;
}
</codeblock>
